CSS MARGIN

Margins are use to create space around elements, outside of any defined borders.

CSS OUTLINE

The css outline is a line drawn outside the element's border.
css has the following outline properties

  • outline style
  • outline width
  • outline color
  • outline offset

note: outline differs from borders! unlike border, the outline is drawn outline the elements border, and may overlap other content.
