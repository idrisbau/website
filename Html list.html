An unordered list

An unordered list start with the ul tag.

  • coffee
  • tea
  • milk

Ordered list

An ordered list starts with OL tag.

  1. Coffee
  2. Tea
  3. Milk

HTML Description list

Html also supports description lists. A description list is a list of terms, with a description list.

COFFEE
_black hot drink
MILK
_white cold drink

Nested html

Lists can be nested(list inside list):

  • coffee
  • Tea
    • Black tea
    • Green tea
  • Milk
